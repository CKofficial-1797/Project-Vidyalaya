# Project Vidyalaya

<p align="center"><b>Project Vidyalaya &copy; 2026</b></p>

## Overview

Project Vidyalaya is a full-featured online learning platform built for scalable course delivery, secure content access, and real-time user engagement.  
It is designed to serve both learners and administrators with a structured, interactive, and production-ready educational ecosystem.

The platform emphasizes performance, security, and maintainability while supporting paid content and live interactions.

---

## Tech Stack

**Backend:** Node.js, Express.js, TypeScript  
**Frontend:** Next.js, TypeScript, Tailwind CSS  
**Database:** MongoDB  
**Caching:** Redis  
**Real-time Communication:** Socket.IO  
**Payments:** Stripe  
**Video DRM:** VideoCipher

## Features

### Learner Features
- Secure registration and login using Email, Google, or GitHub
- OTP-based verification for enhanced account security
- Profile management and account settings
- Course discovery with search and filtering
- Secure course purchases using Stripe
- Email confirmation after successful transactions
- Structured courses with sections and lessons
- Interactive Q&A and review system
- Real-time notifications for admin responses
- DRM-protected video streaming
- Learning progress tracking and certificate downloads

### Administrator Features
- Centralized admin dashboard
- Real-time analytics for users, courses, and orders
- Course creation and content management
- Video, section, and resource organization
- Admin team and permission management
- User profile and activity management
- Website customization (hero section, FAQs, banners, categories)
- WebSocket-based real-time notifications
- Content moderation for questions and reviews
